# Project Roadmap - Future Features

*   **Pagefind Site-wide Search**: Integrate Pagefind to support fast static full-text search across all 600+ static pages.
*   **Curriculum Progress Tracker**: Implement client-side localStorage checks to let users mark lessons as completed and track progress.
*   **Per-Term Canonical Citations**: Complete the rewrite of the remaining 66 terms and add a `sources[]` array referencing specs or original documentation.
*   **Deploy to Production**: Deploy the compiled static site to Netlify once all glossary terms have been successfully rephrased.
*   **Automated Sync Pipeline**: Set up a daily GitHub Action to sync content changes from upstream repositories and automatically create pull requests.
*   **Interactive Concept Graph**: Create a D3.js or SVG concept map visualizing how terms, curriculum lessons, and domains link together.
*   **Interactive Quizzes**: Add client-side multiple-choice questions for each curriculum lesson to test user knowledge.
