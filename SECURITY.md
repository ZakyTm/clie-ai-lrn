# Security Policy

## Supported Versions

| Version | Supported |
| ------- | --------- |
| Latest  | ✓         |

## Reporting a Vulnerability

Please **do not** open a public GitHub issue for security vulnerabilities. Instead, please report vulnerabilities via email:

**Email**: zakaria.toumi.zt@gmail.com

Please include:
*   A description of the vulnerability.
*   Detailed steps to reproduce the issue.
*   The potential impact and severity.
*   A suggested fix (optional).

You will receive a response within 48 hours. 

As this project is a static website hosted on Cloudflare Pages with no custom backend, database, or user authentication system, the scope of security is limited to build-time pipeline security, dependency audits, and overall content integrity.
